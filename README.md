# static-site-generator

This project was built via a step-by-step course provided by boot.dev

The purpose of this project is to create a static HTML site based on markdown files. Built with Python.

## getting started

### prerequisite
- must have Python runtime installed 

### building site
1. Pull down the repo to your local pc
2. With the repo folder, run the following command to build a local version
   - ```./main.ph```
3. Site can then be viewed locally at http://0.0.0.0:8888/
